# 🔥 AI Virality Analyzer

An AI-powered tool that analyzes content and predicts its viral potential.

## 🚀 Features
- Virality score (0–100)
- Hook analysis (first 3 seconds)
- Pacing and engagement analysis
- Caption optimization
- Competitor comparison
- Suggested hashtags
- Trending audio recommendations
- Improved content version

## 🛠️ Tech Stack
- Python
- Streamlit
- Matplotlib
