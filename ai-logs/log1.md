## Prompt
How to build an AI virality analyzer using Streamlit?

## Response
Use Streamlit to create a web interface where users can input content. Apply scoring logic based on factors like hook strength, emotional words, and engagement signals, then display the results with charts.

---

## Prompt
How to calculate virality score?

## Response
Break the score into components such as hook, emotion, engagement, length, and audience targeting. Assign weights to each and combine them into a final score out of 100.

---

## Prompt
How to improve content virality?

## Response
Use strong hooks, emotional triggers, short and engaging text, and ask questions. Add trending hashtags and audio suggestions to increase reach.

---

## Prompt
How to deploy Streamlit app?

## Response
Upload the project to GitHub and deploy it using Streamlit Cloud by connecting the repository and selecting the main app file.

## Prompt
How to add charts in Streamlit?

## Response
Use matplotlib or built-in Streamlit charts to show score breakdown with bar graphs.

---

## Prompt
How to improve UI in Streamlit?

## Response
Use custom CSS, proper layout, headings, and sections to make the app look professional.
